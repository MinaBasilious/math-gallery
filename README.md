## Posting

Posts are written and published by the project author.

If you would like to:

- suggest a post idea
- report a correction or review for an existing post

please open an issue first to discuss it.

For the technical details of how posts are structured, see the
[Posting Guide](./POST_GUIDE.md).

## Contributing

If you want to add to the art gallery or report a bug, feel free to open an issue

Check out the [development guide](./DEVELOPMENT.MD) for details on contributing.
